# Student AI Tools Usage Analysis

This project explores a dataset related to how college students use various Artificial Intelligence (AI) tools in their academic and personal lives. 

## Overview
The `final.ipynb` Jupyter Notebook contains an end-to-end data analysis workflow, including:
- **Data Loading and Cleaning**: Handling missing values, standardizing column formats, and preparing the dataset for analysis.
- **Exploratory Data Analysis (EDA)**: Profiling the data to understand distributions of key variables.
- **Statistical Summaries**: Analyzing daily usage hours, trust in AI tools, and their perceived impact on grades.
- **Data Visualizations**: Generating visual insights into students' preferred AI tools, use cases (like assignments, coding help, content writing), and overall awareness levels across different streams and states.

## Dataset Features
The dataset contains information on several variables, including:
- Student demographics and college information (`Stream`, `Year_of_Study`, `State`)
- AI tool preferences (`ChatGPT`, `Gemini`, `Copilot`, etc.)
- Usage patterns (`Daily_Usage_Hours`, `Use_Cases`)
- Perceptions and outcomes (`Trust_in_AI_Tools`, `Impact_on_Grades`, `Awareness_Level`)
- Accessibility (`Device_Used`, `Internet_Access`)

## Getting Started
To view the analysis, simply open the `final.ipynb` notebook. Ensure you have the required Python dependencies installed (e.g., `pandas`, `matplotlib`, `seaborn`) to run the code cells and render the visualizations locally.
